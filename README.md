# 🗺️ LlanquihueTourApp

## 📌 Descripción

Aplicación desarrollada en Java para gestionar entidades y servicios turísticos de la agencia Llanquihue Tour utilizando Programación Orientada a Objetos.

---

## 📁 Estructura

```
src
├── model
├── data
├── util
├── ui
└── resources
```

---

### model
- Persona

- Direccion
- Rut
- Registrable
- ServicioTuristico
- RutaGastronomica

- GestorServicios

### util

### ui
- Main

2. Ejecutar la clase `Main`.
